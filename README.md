# superstore-sales-performance-dashboard
End-to-end data analysis project using Excel, SQL, and Tableau
Superstore Sales Performance Dashboard

📊 Project Overview

This project analyzes retail sales data to uncover insights into revenue, profitability, and regional performance. The analysis was performed using Excel, SQL, and Tableau.

🛠 Tools & Technologies

Excel (data cleaning & pivot analysis)

SQL (data querying & aggregation)

Tableau (interactive dashboard)

📈 Key Insights

Total revenue generated: $2.3M

Total profit: $286K with a 12.5% profit margin

West region has the highest sales performance

Technology is the most profitable category

South region shows the lowest sales performance

🔍 SQL Example
SELECT region, SUM(sales) AS total_sales
FROM superstore
GROUP BY region
ORDER BY total_sales DESC;
